# Inverted Search in C

A C-based project that mimics search engine indexing by creating an inverted index of words across multiple text files. Each unique word maps to a list of files where it occurs, allowing fast retrieval of files containing a queried word.

---

## Features
- Builds inverted index using singly linked lists
- Each word node maintains a sublist of files containing it
- Avoids duplicates for words and files
- Case-insensitive search
- Validates input files before processing

---

## Technologies Used
- C Programming
- File Handling
- Linked Lists
- String Manipulation
